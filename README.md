## PokemonApp

Приложение, в котором отображается список покемонов и детальная информация о каждом персонаже

<b>Стек:</b>
<p>Clean Architecture, Retrofit, OkHttp, Dagger 2, Coroutines, MVVM, Navigation Architecture Component, Glide</p>
